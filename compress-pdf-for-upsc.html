UPSC Document Upload Rules

  • File Limit: 20KB to 300KB (for certificates like OBC/SC/ST, Domicile).
  • Format: PDF only.
  • Quality: Must be legible. (This tool ensures text remains sharp).
  • Security: Do not upload password-protected PDFs.

How to Compress PDF for UPSC

  4. Download and upload to the UPSC OTR portal.

Frequently Asked Questions

What is the PDF size limit for UPSC documents?

For UPSC OTR and application forms, scanned documents (PDF) must generally be between 20KB and 300KB. Some specific forms allow up to 1MB, but 300KB is the standard for certificates.

Does this tool make the text unreadable?

No. Since UPSC allows up to 300KB, this tool uses a high-quality compression setting that keeps text sharp while still reducing the file size significantly.

Can I compress multiple pages?

Yes, this tool supports multi-page PDFs. It processes each page individually and combines them into a single optimized PDF file.
