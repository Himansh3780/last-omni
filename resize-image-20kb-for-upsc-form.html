UPSC Photo Requirements (Civil Services / NDA)

  • Size Range: 20KB (Minimum) to 300KB (Maximum).
  • Dimensions: 350px width x 350px height (Square).
  • Format: JPG / JPEG.
  • Details: Name and Date of photo taken must be on the image.

Frequently Asked Questions

What are the photo dimensions for UPSC?

UPSC requires the photograph to be 350 pixels width x 350 pixels height (Square format).

What is the size limit for UPSC photo upload?

The file size must be between 20KB (minimum) and 300KB (maximum). Files smaller than 20KB are rejected.

Is the name and date required on UPSC photo?

Yes, most recent UPSC notifications require the candidate's name and the date the photo was taken to be printed on the image.
